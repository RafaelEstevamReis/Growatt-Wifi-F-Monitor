# Known values for entities

## System Status: growatt_System_Status

Observed:
* 2: Using Battery
* 5: Using Solar
* 11: Using AC

Growatt modbus protocol PDF:
* 0: Standby;
* 1; （No Use）
* 2: Discharge;
* 3: Fault;
* 4: Flash;
* 5: PV charge;
* 6: AC charge;
* 7: Combine charge;
* 8: Combine charge and Bypass;
* 9: PV charge and Bypass;
* 10: AC charge and Bypass;
* 11: Bypass;
* 12: PV charge and Discharge;
